## Trees
Implementation of the following trees:
- binary trees
- avl trees
- splay trees

### Build
```shell
export CC=/usr/bin/gcc
export CXX=/usr/bin/g++
cmake -S . -B ./build
cmake --build build
```

### Run
```shell
./build/binary-search-tree
./build/avl-tree
./build/splay-tree
```
