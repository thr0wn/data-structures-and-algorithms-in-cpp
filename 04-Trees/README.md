## Trees
Implementation of binary and avl trees.

### Build
```shell
export CC=/usr/bin/gcc
export CXX=/usr/bin/g++
cmake -S . -B ./build
cmake --build build
```

### Run
```shell
./build/trees/binary-search-tree
```
