## Trees
Implementation of the following trees:
- binary tree
- avl tree
- splay tree

## Maps and Sets
Implementation of:
- Map demo
- Word ladder

### Build
```shell
export CC=/usr/bin/gcc
export CXX=/usr/bin/g++
cmake -S . -B ./build
cmake --build build
```

### Run
```shell
./build/binary-search-tree
./build/avl-tree
./build/splay-tree
./build/map-demo
./build/word-ladder
```
